

 Overview
This project is an expert system for car fault diagnosis, developed during my current Master 1 year.  
It uses rule-based reasoning to infer potential car problems based on user input.

---

 Features
- Welcome Screen – introduces the system to the user.  
- Diagnosis Screen – interactive sequence of yes/no questions. Each answer triggers the next relevant question according to the defined rules.  
- Result Screen – displays the final **diagnosis**, **possible problem**, and the **rules that were triggered**.

---

## Methodology
- Rules-based inference system (Forward chaining based system): the system evaluates user responses against a predefined knowledge base.  
- Demonstrates understanding of expert systems, AI logic, and problem-solving workflows.
